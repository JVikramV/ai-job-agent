# 🚀 AI Job Applier

Autonomous AI agent that navigates real websites and applies to jobs automatically.

## 🔥 Features
- Multi-step web automation (TinyFish)
- Job scraping + filtering
- Form filling simulation
- Live execution logs
- Success tracking

## 🧠 Tech Stack
- FastAPI (Backend)
- React + Vite (Frontend)
- TinyFish Web Agent API

## 🎯 Problem Solved
Applying to jobs is repetitive and time-consuming. This agent automates the process end-to-end.
